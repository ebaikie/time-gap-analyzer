Privacy Policy

Last updated: 13 June 2026

Summary: Time Gap Analyzer does not store, sell, or share any data. Your screenshot is processed in server memory and discarded immediately after the results are returned to your browser.

What the app does

When you upload a screenshot, it is sent from your browser to the server, analysed using a locally-installed Tesseract OCR engine, and a summary of time gaps, overlaps, and pay-requirement checks is returned to your browser. No data is retained after the response is sent.

Data handling

  • Screenshots are held in server memory only for the duration of the analysis (typically under a second), then discarded. They are never written to disk, never retained between requests, and never transmitted to any third party.
  • Timesheet times read from the screenshot exist only in memory during analysis and are discarded when the response is sent.
  • Settings (required start time, minimum logged hours) are sent as form fields with each request. They are not stored on the server — you set them fresh each session.

Network activity

The server makes no outbound network requests. All OCR processing uses a locally-installed Tesseract engine — there are no external APIs, no model downloads, and no CDN calls. Your screenshot never leaves the server it was sent to.

Third parties

The app does not use analytics, advertising, tracking, or any third-party data-collection services.

Data sale and transfer

No user data is sold or transferred to any third party, used for any purpose unrelated to analysing your screenshot, or used to determine creditworthiness or for lending purposes.

Browser extension

The companion browser extension has a separate privacy policy — its data handling differs because OCR runs locally in your browser rather than on a server.
